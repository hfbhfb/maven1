



## maven下载和配置
- 下载
   - https://maven.apache.org/download.cgi
- 配置环境变量
   - MAVEN_HOME
   - PATH



## maven基础概念

- 仓库
   - 本地仓库，远程仓库（中央仓库，私服）
- 坐标
   - https://repo1.maven.org/maven2/
- groupId: 定义当前Maven项目隶属组织
- artifactId: maven项目名称
- version: 定义当前项目版本号
